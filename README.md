# 📚 Design Patterns - Repositório de Estudos

## 🎯 Objetivo

Este repositório foi criado com o intuito de servir como base de estudo para a disciplina de Design Patterns do 4º semestre. Aqui serão reunidas implementações práticas e explicações detalhadas dos padrões de projeto, com foco no aprendizado contínuo e na consolidação de conceitos fundamentais de engenharia de software.

A proposta é ir além da teoria, aplicando cada padrão em cenários reais ou simulados, entendendo quando utilizar, quais problemas resolve e quais são seus impactos na arquitetura do sistema.

---

## 🧠 Conteúdo

O repositório contempla os **23 Design Patterns clássicos (GoF)**, organizados por categoria:

### 🔹 Creational (Criacionais)

* Abstract Factory
* Builder
* Factory Method
* Prototype
* Singleton

### 🔹 Structural (Estruturais)

* Adapter
* Bridge
* Composite
* Decorator
* Facade
* Flyweight
* Proxy

### 🔹 Behavioral (Comportamentais)

* Chain of Responsibility
* Command
* Interpreter
* Iterator
* Mediator
* Memento
* Observer
* State
* Strategy
* Template Method
* Visitor

Cada padrão contará com:

* 📖 Explicação teórica
* 💻 Implementação prática
* ⚙️ Exemplos de uso

---

## 🧪 Desafios

Na pasta `desafios`, há um conjunto de exercícios práticos para cada Design Pattern.

O objetivo desses desafios é reforçar o aprendizado através da prática, estimulando a resolução de problemas e o desenvolvimento da lógica aplicada aos padrões.

---

## Avaliações

Na pasta `Avaliações`, há um conjunto de avaliações respondidas de semestres passados. O intuito é testar o que se foi aprendido com desefios de nível acadêmico. Como método para acompanhar minha evolução, haverá duas versões de resoluções: 
- uma antes de corrigir, com conhecimento cru
- uma corrigida e revisada com a melhor resolução

## 🚀 Intenção

Este repositório funciona como:

* Um diário de aprendizado
* Um material de revisão
* Um portfólio técnico

Além disso, busca demonstrar a evolução no entendimento de boas práticas, organização de código e padrões de projeto utilizados no desenvolvimento profissional.
